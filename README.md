##`npm run pg:prod`: access to production database
Pre-requisites:
- Login in heroku

In order to run `npm run pg:prod` command one needs to install pg:psql. Follow the next steps:
1. Install [postgres](https://postgresapp.com/)
2. `$ sudo mkdir -p /etc/paths.d && echo /Applications/Postgres.app/Contents/Versions/latest/bin | sudo tee /etc/paths.d/postgresapp`
